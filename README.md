# 工位服刑 · OfficePrison

> 上班如服刑，放风即自由。

一个把「上班」翻译成「服刑」的**纯单机网页玩具**。输入月薪领取你的《劳动判决书》，摸鱼时点「放风」计时，结束获得一张写明「折抵刑期 + 国家赔偿金额」的《减刑裁定书》，下班领取《释放证明》。

严肃的司法文书格式 × 极度荒谬的内容，落差本身就是笑点。它不是效率工具、不帮你自省——它只帮你嘲讽时间，然后生成一张值得截图发出去的图。

## ✨ 核心功能

- **入狱登记**：填昵称、月薪、工时、作息，生成专属《劳动判决书》长图（含随机判决书编号与戏仿印章）。
- **放风计时器**：一键开始放风，实时跳动「国家赔偿累计 ¥XX.XX」，轮播狱中标语。支持多种放风场景（工位神游 / 如厕 / 茶水间 / 会议走神 / 网上冲浪 / 园区遛弯……）。
- **减刑裁定书**：放风结束按「收监」，即时生成裁定书——折抵刑期、依法获得的国家赔偿金额、按时长与场景分桶的随机锐评。
- **今日释放**：手动「申请释放」，汇总今日服刑时长、放风次数、累计赔偿，生成《释放证明书》。
- **服刑档案**：累计服刑天数、历史总国家赔偿、单次放风最长纪录，一键销毁全部数据。
- **一键保存长图**：所有文书用 [html2canvas](https://html2canvas.hertzen.com) 生成带水印的长图，方便截图分享。

## 🔒 隐私

- **纯前端单机**：无账号、无后端、无数据库。
- 所有数据（月薪、放风流水）**仅存于用户本地 localStorage**，永不上传。
- 提供「销毁全部档案」一键清空本机数据。

## 🛠 技术栈

- 单文件原生 HTML + CSS + JavaScript，零构建。
- `html2canvas` 本地自托管（`html2canvas.min.js`），不依赖外部 CDN。
- [Microsoft Clarity](https://clarity.microsoft.com) 埋点，仅采集匿名行为数据用于产品分析（**不采集月薪等个人信息**）。

## 📁 项目结构

```
├── index.html            # 全部页面与逻辑（单文件应用）
├── html2canvas.min.js    # 本地长图生成库
└── README.md
```

## 🚀 本地运行

直接用浏览器打开 `index.html` 即可，无需任何依赖或构建步骤。

## ☁️ 部署

纯静态站，可部署到任意静态托管（Vercel / Cloudflare Pages / Zeabur / 对象存储等）。根目录首页为 `index.html`，需与 `html2canvas.min.js` 一同部署。
