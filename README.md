# 工位服刑 · OfficePrison

> 上班如服刑，放风即自由。

一个把「上班」翻译成「服刑」的**纯单机网页抽象工具**。输入月薪领取你的《劳动判决书》，摸鱼时点「放风」实时累计劳改津贴，需要专注时进入「沉浸式劳改」番茄钟，下班领取《释放证明》，每周还能导出一份《改造周报》。

严肃的司法文书格式 × 极度荒谬的内容，落差本身就是笑点。它不帮你自省、也不追求数据精确——它把打工人的一天翻译成一场荒诞的服刑，顺手生成一张张值得截图发出去的图。

## ✨ 核心功能

- **入狱登记**：只需一个月薪（默认预填、可改）即可入狱，其余作息项均有默认值；生成专属《劳动判决书》长图，含随机判决书编号、戏仿印章与扫码入狱二维码。
- **放风计时**：一键开始放风，实时跳动「劳改津贴」，多种放风场景（工位神游 / 如厕 / 茶水间 / 会议走神 / 网上冲浪 / 园区遛弯……）随时切换。
  - **跨退出续计**：退出网页/锁屏后再打开，计时继续，离开的时间也照算。
  - **单次上限**：超过 4 小时自动收监（疑似越狱），防止忘记收监导致数字失真。
- **收监回执**：每次放风结束，即时给出折抵刑期、劳改津贴与随机锐评。
- **沉浸式劳改（番茄钟）**：监狱牢房风格的专注模式——牢门铁栏、狱警不定时巡查并留下一句训话；可选劳改场景（撰写悔过材料 / 与外方势力周旋 / 列席提审大会……）与沉浸时长（15/25/45/60 分钟）。
- **今日释放**：手动「申请释放」，汇总今日服刑时长、放风次数、劳改津贴，生成《释放证明书》。
- **改造周报**：一份「牛马周报 PPT」样式的数据图——KPI 卡片、放风场景分布条形图、上级批示，专业商业幻灯片外观。
- **服刑档案**：累计服刑天数、历史劳改津贴、单次放风最长纪录；支持随时**修改信息**（改月薪后历史收益按新标准动态重算）。
- **一键保存长图**：所有文书 / 周报均可用 [html2canvas](https://html2canvas.hertzen.com) 生成长图，底部带扫码入狱二维码，方便截图分享。

## 🔒 隐私

- **纯前端单机**：无账号、无后端、无数据库。
- 所有数据（月薪、放风流水）**仅存于用户本地 localStorage**，绝不上传。
- 提供「销毁全部档案」一键清空本机数据。

## 🛠 技术栈

- 单文件原生 HTML + CSS + JavaScript，零构建。
- `html2canvas` 本地自托管，长图生成不依赖外部 CDN。
- 自托管**思源宋体子集**（`NotoSerifSC-subset.woff2`），全设备统一公文字体，手机端不再回退成黑体。
- 内嵌二维码（静态生成），扫码直达站点。
- [Microsoft Clarity](https://clarity.microsoft.com) 匿名行为埋点，用于分析使用路径（**不采集月薪等个人信息**）。

## 📁 项目结构

```

├── NotoSerifSC-subset.woff2 # 自托管宋体子集
└── README.md
```

## 🚀 本地运行

直接用浏览器打开 `index.html` 即可，无需任何依赖或构建步骤。

## ☁️ 部署

纯静态站，可部署到任意静态托管（Vercel / Cloudflare Pages / Zeabur / 对象存储等）。根目录首页为 `index.html`，需与 `html2canvas.min.js`、`NotoSerifSC-subset.woff2` 一同部署。以后更新只需推送到 GitHub，托管平台自动重新部署。
